🧩 LOS 3 COMPONENTES QUE NECESITAS
Componente	Dónde corre	Qué hace
🧠 Base de datos MySQL	Tu PC (local)	Guarda los datos de los pacientes
⚙️ API Flask (api_sanalter.py)	Tu PC (local)	Recibe los datos de la app y los guarda en la BD
📱 App Kivy (paciente_app_kivy.py)	Tu celular	Pantalla para capturar los datos y enviarlos a la API
